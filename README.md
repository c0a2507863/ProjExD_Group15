# 強制横スクロール系ゲーム

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 主人公キャラクターをジャンプ操作により，．．．

## ゲームの遊び方
* スペースキーでキャラクターをジャンプ操作し、ハイスコアを目指す
* 障害物に当たったらゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 自動横スクロール、SPACEジャンプ、重力、障害物、当たり判定、GAME OVER

### 分担追加機能
* ステージギミック(動く障害物・穴・足場)
* アイテム(障害物消去・無敵)
* 一時停止・再開・リスタート機能
* 二段ジャンプ:ジャンプ中にもう一度スペースキーを押すことで、二段ジャンプができるようになった。（三段ジャンプ以上はできない）
* ターボ:シフトキー長押しで、ロケットのような上方向のターボを出せるようになった。
* エフェクト・サウンド・デザイン系
